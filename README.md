# Phonebook API

## Base URL

https://phonebookapp-o18l.onrender.com

## Endpoints

### Get all persons

**GET** `/api/persons`

### Get a person by ID

**GET** `/api/persons/:id`

### Delete a person by ID

**DELETE** `/api/persons/:id`

### Add a new person

**POST** `/api/persons`

### Info

**GET** `/info`
